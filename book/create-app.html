构建第一个 ember 应用程序

构建 ember 应用程序

构建 ember app 使用官方出品的构建工具 ember-cli

npm install -g ember-cli@3.28

安装完成之后，创建一个空文件夹 my-app，然后进入执行初始化

mkdir my-app
cd my-app
ember init

等待安装完成，然后启动 server

ember serve
